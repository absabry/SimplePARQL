To launch the application
Open IntelliJ
src > source > webservice > Server Launch this class to open the Java server 

Go to src > clientwebservice > server.js and launch this js file on node.js environement to open the server client

Go to src > clientwebservice > Interface
open simpleparql.html to open the user interface


Note :
To change the constants of the web inferface :
    The websocket.js contain the connecting port
    simleparql.html file contain the version of used API, in the script head

To change any other constants :
    The query's constants, base's constants or even Java Server constan's :  you can check the configuration file for each.
    You can find the configuration rep beside the README file
